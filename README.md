# CyberJean.github.io 

A daily cybersecurity blog powered by an AI news agent, built to stay sharp, 
stay informed, and show recruiters what I'm made of.

---

##  About This Project

CyberJean.github.io is a living portfolio and daily cybersecurity blog. Every 
morning, an AI-powered agent automatically researches and publishes the top 
cybersecurity news stories — covering threats, vulnerabilities, data breaches, 
and industry trends. Beyond the news, this site documents my personal learning 
journey, the projects I'm building, and the skills I'm developing as I grow in 
the cybersecurity field.

This isn't just a blog — it's proof of work.

---

##  How It Works

1. **GitHub Actions** runs a scheduled workflow every day at midnight UTC
2. The workflow calls the **Google Gemini AI API** to research and write a 
   professional cybersecurity briefing
3. The generated post is automatically committed to the repository as a 
   Jekyll markdown file
4. **GitHub Pages** rebuilds and deploys the site automatically
5. A new post is live every morning — no manual effort required

## 🚀 Running Locally
```bash
# Clone the repo
git clone https://github.com/CyberJean98/CyberJean.github.io.git

# Install dependencies
bundle install

# Run locally
bundle exec jekyll serve
```

Then visit `http://localhost:4000` in your browser.
